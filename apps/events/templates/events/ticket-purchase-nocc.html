Realice una transferencia bancaria indicando los siguientes datos: - **Banco**: {{ pythoncanarias.bank}} - **IBAN**: {{ pythoncanarias.iban }} - **Importe**: {{ article.price }} € - **Concepto**: Entrada para {{ article.event.slug }} - **Beneficiario**: {{ pythoncanarias.name }} 2. Una vez recibamos este email, generaremos su entrada y se la enviaremos por correo electrónico como fichero adjunto PDF. ## PayPal

No es necesario enviar correo electrónico, ya que PayPal se encarga de notificar el envío de dinero. 3.
